# Remie

話すように書くだけの、いちばん速いリマインダーアプリ（iOS）。
*The fastest way to set a reminder — just write it the way you'd say it.*

「明日の朝9時にミーティング」と打つだけで、Remie が日時を自動で読み取ります。
ピッカー操作も余計なタップも不要。完全オフラインで、データは一切収集しません。

## 主な機能
- 自然文入力（「明日」「3日後」などの相対表現も自動変換）
- 保存前に内容を確認・修正
- 繰り返しリマインダー（毎日・毎週・毎月）
- ホーム画面ウィジェット
- 時間ぴったりのローカル通知 / 日本語・英語対応

## プライバシー
完全オフライン。アカウント不要・トラッキングなし・データ収集なし。
[プライバシーポリシー](https://kotapremium.github.io/remie/privacy.html)
